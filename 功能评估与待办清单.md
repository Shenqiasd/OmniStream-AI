### ⚠️ 需要配置的功能

#### 1. 平台 OAuth 配置（必须）

**需要配置的平台**：

```bash
# TikTok
TIKTOK_CLIENT_ID=
TIKTOK_CLIENT_SECRET=
TIKTOK_REDIRECT_URI=http://localhost:3002/callback/tiktok

# Twitter/X
TWITTER_CLIENT_ID=
TWITTER_CLIENT_SECRET=
TWITTER_REDIRECT_URI=http://localhost:3002/callback/twitter

# YouTube
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REDIRECT_URI=http://localhost:3002/callback/youtube

# Meta (Facebook/Instagram/Threads)
META_FACEBOOK_CLIENT_ID=
META_FACEBOOK_CLIENT_SECRET=
META_INSTAGRAM_CLIENT_ID=
META_INSTAGRAM_CLIENT_SECRET=
META_THREADS_CLIENT_ID=
META_THREADS_CLIENT_SECRET=

# LinkedIn
META_LINKEDIN_CLIENT_ID=
META_LINKEDIN_CLIENT_SECRET=

# Pinterest
PINTEREST_CLIENT_ID=
PINTEREST_CLIENT_SECRET=
```

**获取方式**：
- TikTok: https://developers.tiktok.com/
- Twitter: https://developer.twitter.com/
- YouTube: https://console.cloud.google.com/
- Meta: https://developers.facebook.com/
- LinkedIn: https://www.linkedin.com/developers/
- Pinterest: https://developers.pinterest.com/

---

#### 2. 文件存储配置（推荐）

**AWS S3 配置**：

```bash
AWS_S3_REGION=
AWS_S3_BUCKET=
AWS_S3_ACCESS_KEY_ID=
AWS_S3_SECRET_ACCESS_KEY=
```

**替代方案**：
- 使用本地文件存储（需要修改代码）
- 使用阿里云 OSS（需要修改代码）
